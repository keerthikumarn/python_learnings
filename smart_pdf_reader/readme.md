<p id="description">Ollama and Python based app to scan and read any uploaded PDF docs</p>

  
  
<h2>🧐 Features</h2>

Here're some of the project's best features:

*   📄 PDF Upload: Upload any document in PDF format
*   💬 Interactive Chat: Ask questions and have a conversation about the document
*   🎓 Step-by-Step Learning: The AI tutor guides you through concepts incrementally 🔄 Clear Chat: Reset the conversation to start fresh
*   🤖 Powered by Ollama model: Uses qwen2.5-coder:7b for intelligent responses

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
https://github.com/keerthikumarn/python_learnings.git
```

<p>2. Navigate to smart_pdf_reader</p>

```
cd smart_pdf_reader
```

<p>3. Install the requirements</p>

```
pip install -r requirements.txt
```

<p>4. Run the app</p>

```
streamlit run app.py --server.port 8099
```

<p>5. Launch the app</p>

```
http://localhost:8099
```

<p>6. Browse and Upload any PDF doc</p>

<p>7. Start posting the questions in the chat area</p>
